HyperText Markup Language - HTML

Uma linguagem para estruturar o seu site, uma base básica, ela é estática, usada principalmente para textos, podemos utilizar diferentes tamanhos de parágrafos dentro do html, quebra de páginas, não é utilizada para estilização, fazer cabeçalhos, roda pés, entre diversas utilizações.

HTML

Representa a raiz de um HTML, todos elementos devem ser html

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplo:

<!DOCTYPE HTML> 
<html>
    <head> </head>
    <body> </body>
</html>

HEAD

Head, como no nome é a cabeça do seu site, onde podemos colocar title, como titulo do seu site, description, como a descrição do seu site e as keywords, para deixar seu site no topo das pesquisas, link para scripts entre outras ações.

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplo:

<html> 
  <head>
    <title>Título do Documento</title>
  </head> 
</html>

TITLE

A tag title, define um título para o seu documento aparecendo assim na barra superior do seu navegador

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplo:

<title> título para aba do navegador </title>

META

Ele define qualquer informação de metadados.

Atributos:

  • Name: efine o nome do document-level metadata.
  • Charset: Define qual escolha de caracteres será usada na página.
  • Http-equiv: Define a pragma isso altera o comportamento de servers e useragents.

Exemplos:

<meta name="author" content= "Marcel Leite de Farias">

BODY

Tudo que vai no corpo do seu site, o conteúdo dele.

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<body>
  <b>Olá</b>
</body>

HEADER

Normalmente usamos o <header> </header> para cabeçalhos, como o Título da sua página.

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<header>
  <h1>Título da página</h1>
<header>

FOOTER

A tag <footer> </footer> é usada para rodapé, normalmente usada para informações do autor da página.

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<footer>
  Direitos reservados Marcel Leite de Farias - 2021
</footer>

H1 ATÉ H6

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<h1> Cabelhaço de nível 1 </h1>
<h2> Cabelhaço de nível 2 </h2>

  • Não foram vistos atributos para essa tag.

SPAN

Essa tag pode ser usada para agrupar elementos afins de estilo.

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<p><span> Algum texto </span></p>

PRE

É uma tag para utilizar textos pré-formatados.

Atributos:

  • wrap: é utilizada para especificar se o navegador pode ou não quebrar linhas dentro das palavras.

Exemplos:

<pre> 
Michael Jackson Billie Jean

She was more like a beauty queen from a movie scene
I said don't mind, but what do you mean, I am the one
Who will dance on the floor in the round?
She said I am the one, who will dance on the floor in the round
</pre>

STRONG

Deixa tudo que está dentro dessa tag em negrito.

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<p> Saveiro cross <strong> é linda </strong> <p>

TAG EM

A tag EM marca o texto que tem ênfase

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<p> 
  No rock o cantor <em> Michael Jackson </em> era o rei do pop/rock agora é o <em>Pedrinho</em>
</p>

TAG B

A tag b geralmente é usada para destacar palavras, nomes, nomes de produtos, por representar um intervalo de texto diferente do texto normal.

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<p> O supermercado KUNS está com esses produtos em desconto <b> arroz </b> e <b> feijão </b>

TAG BR

Essa tag possibilita uma quebra de linha em um texto, é bastante útil para textos, poemas, ou endereços

  • Não foram vistos atributos para essa tag.

Rua Expedicionário Anibal dos Passos <br>
89225-676 BR

TAG HR

Representa uma quebra de temática com uma linha horizontal.

Atributos:

  • Color: Define uma regra de cor através da cor, ou um valor hexadecimal

Exemplos:

<p> Como eu amo programar no Visual Studio Code </p> 
<hr>
<p> Como eu consigo ser tão rápido no aprendizado? </p>

TAG UL

Essa tag representa uma lista desordenada, sem ordem rígida, uma sequencia de itens que não trazem uma ordenação númerica e as suas posições

Atributos:

  • Type: Usado para estabelecer o marcador da lista.

Exemplos:

<ul>
  <li> Item 1 </li>
  <li> Item 2 </li> 
</ul>

TAG OL

Essa tag representa uma lista ordenada, esses itens ordenados em uma lista são mostrads com uma contagem que os precede.

Atributos:

  • Type: Indica o tipo de númeração, seja com números ou letras.

Exemplos:

<ol>
  <li> Primeiro <li> 
  <li> Segundo <li> 
<ol>

TAG LI

É usada para representar um item dentro de uma lista

Atributos:

  • Type: Serve para indicar o tipo de numeração

Exemplos:

<ul>
  <li> Primeiro item do li </li>
  <li> Segundo item do LI </li>
<ul>

TABLE

Representa dados em duas dimensões ou mais, utilizada para criação de tabelas.

Atributos:

  • Border: Define os pixels, o tamanho do quadro ao redor.

Exemplos:

<table> 
  <tr>
    <td> John </td>
    <td> Doe </td>
  </tr>
  <tr>
    <td> Luca </td>
    <td> Doe </td>
  </tr> 
</table>

THEAD

Define um conjunto de linhas que definem o cabeçalho das colunas da tabela.

Atributos:

  • Align: Esse atributo enumerado especifica como o alinhamento horizontal de cada conteúdo de célula será tratado.
  • Bgcolor: Este atributo define a cor de fundo de cada célula da coluna.

Exemplos:

<table>
  <thead> 
    <tr> 
      <th colspan = "2" > A mesa </th> 
    </tr> 
  </thead>
</table>

TBODY

Essa tag encapsula um conjunto de linhas da tabela, indicando que eles compreender o corpo da tabela

Atributos:

  • Bgcolor: A cor de fundo da tabela.
  • Align: Este atributo enumerado especifica como o alinhamento horizontal de cada conteúdo de célula será tratado.

Exemplos:

<table> 
  <caption>Orçamento do conselho (em $) 2018 </caption>
  <thead> 
    <tr>
      <th>Itens</th>
      <th> scope = "col">Despensa</th> 
    </tr> 
  </thead> 
  <tbody> 
    <tr> 
      <th scope = "row"> Donuts </th> 
      <td> 3,000</td> 
    </tr> 
    <tr> 
      <th scope="row">Papelaria</th>
      <td>18,000</td>
    </tr>
  </tbody>
</table>

TFOOT

A tag tfoot define um conjunto de linhas as quais farão parte do rodapé

Atributos:

  • Brcolor: Este atributo define uma cor de fundo para cada célula

Exemplos:

<table> 
  <tfoot> 
    <tr>
      <th scope="row">Totals</th>
      <td>21,000</gt>
    </tr>
  </tfoot>
</table>

TAG TR

Essa tag define uma linha de células em uma tabela.

Atributos:

  • Bgcolor: A DOMString especificando uma cor a ser aplicada aos planos de fundo de cada uma das células da linha.

Exemplos:

<table>
  <tr>
    <td>Madara Uchiha</td>
    <td>Hashirama</td> 
    <td><time datetime="2012 - 07 - 23">July 23, 2012%</time></td>
    <td>15.00</td>
  </tr>
</table>

TAG TH

Essa tag define uma célula cabeçalho do grupo de células de sua tabela.

Atributos:

  • Bgcolor: Esse atributo define a cor de fundo de cada célula na coluna.
  • Scope: Este atributo define as células a que o cabeçalho (definido no elemento <th>) se relaciona.

Exemplos:

<table>
  <caption>Alien football stars</caption>
  <tr>
    <th scope="col">Player</th>
    <th scope="col">Gloobles</th>
    <th scope="col">Za'taak</th> 
  </tr>
</table>

TAG TD

Essa tag define uma célula de uma tabela que contém dados.

Atributos:

  • Bgcolor: Este atributo define a cor de fundo de cada célula em uma coluna.

Exemplos: 

<table>
  <tr>  
    <th scope="row">TR-7</th>
    <td>7</td>
    <td>4,569</td>
  </tr> 
</table>

TAG A

Essa tag é com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de email, ligações na mesma página ou endereço na URL

Atributos:

  • Href: A URL para qual o hiperlink aponta. Seções de páginas com fragmentos URL, pedaços de arquivo de midia

Exemplos:

<body>
  <nav>
    <ul>
      <li><a href="https//google.com">Website</a></li>
      <li><a href"contato.google@gmail.com>Email</a></li>
    </ul>
  </nav>
</body>

TAG IMG

Essa tag representa a inserção de imagem no documento.

Atributos:

  • Src: URL da imagem, este atributo é obrigatório para o <img> elemento.

Exemplos:

    <img src="image.jpg" alt="Minha figura">

  </figure>

FIGCAPTION

Essa tag representa uma legenda associada com uma figura ou ilustração descrita pelo resto dos dados do elemento

Atributos:

  • Não foram vistos atributos para essa tag.

Exemplos:

<body>
  <figure>     
    <img src="/media/cc0-images/elephant-660-480.jpg"
      alt="Elephant at sunset">

TAG FORM

Representa uma seção de um documento que contém controles interativos que permitem ao usuário submeter informação a um determinado servidor web

Atributos:

  • Action: O URI de um programa que processa as informações enviadas por meio do formulário.

Exemplos:

<form action="" method="get">
  <label for="GET-name">Name:</label>
  <input id="GET-name" type="text" name="name">
  <input type="submit" value="Save">
</form>

TAG INPUT

Essa tag é usada para criação de controles interativos para formulários baseados na web para receber os dados do usuário.

Atributos:

  • Type: O tipo de controle a ser exibido. O tipo padrão é text, se este atributo não for especificado.

Exemplos:

<body>             
  <form action="getform.php" method"get">
    nome: <input type="text" name="nome"*>
  </form>
</body>

TAG SELECT

Representa um controle que apresenta um menu de seleções

Atributos:

  • Name: O nome do controle

Exemplos:

<body>              
  <select name="select">
    <option value="valor1">Valor 1</option>
  </select>
</body>

TAG OPTION

É usado para criar um controle que representa um item dentro de um elemento HTML5

Atributos:

  • Value: O conteúdo textual deste atributo representa o rótulo que explica a opção. Se ele não estiver definido, seu valor padrão é o texto contido no elemento.

Exemplos:

<body>
  <select name"options">
    <option value="valor1">Valor 1</option>
  </select>
</body>

TEXTAREA

Representa um controle de edição para uma caixa de texto, útil quando você quer permitir ao usuário informar um texto extenso em formato livre.

Atributos:

  • Name: O nome do controle.

Exemplos:

<body> 
  <textarea id="story" name="story"
               rows="5" cols="33">
It was a dark and stormy night
  </textarea>
</body>
